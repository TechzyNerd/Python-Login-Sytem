# Simple Python Login System

A minimal Python login system that allows users to **register** and **login**. User credentials are stored in a local JSON file (`users.json`). Perfect for beginners learning basic authentication.

---

## Features
- Register new users with a username and password
- Login with existing credentials
- Stores user data in a JSON file
- Simple and easy to understand for learning purposes

---

## Requirements
- Python 3.x
- No additional libraries required (uses Python's built-in `json` module)

---

## How to Run
1. Clone or download this repository.
2. Make sure `users.json` exists in the same folder (it will be created automatically if not).
